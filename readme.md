# 校园卡管理系统

## 项目简介

本项目为一款基于C语言的模拟校园卡管理系统，支持学生和管理员两类用户，涵盖校园卡充值、消费、上机记录、挂失、密码管理等常用功能。适用于课程设计、C语言学习与实践。

## 功能特性

- **学生功能**：
	- 登录/登出
	- 校园卡余额查询与消费
	- 上机时长选择与记录
	- 上机历史查询与统计
	- 密码修改
	- 挂失/解挂
- **管理员功能**：
	- 学生信息查询
	- 校园卡充值
	- 重置学生密码
	- 数据维护
- **安全性**：
	- 密码输入安全读取
	- 账户状态（正常/挂失/冻结）管理

## 编译与运行

### 环境要求
- Windows 平台（依赖 <windows.h>）
- CMake 3.10 及以上
- 支持 C11 的 C 编译器（如 MSVC、MinGW）

### 编译步骤
1. 安装 CMake 并配置好 C 编译器。
2. 在项目根目录下执行：
	 ```sh
	 mkdir build
	 cd build
	 cmake ..
	 cmake --build .
	```
3. 生成的可执行文件为 `School_Card_Manage`，可直接运行。

### 运行说明
- 首次运行会自动生成数据文件。
- 管理员初始密码为 `123456`。
- 用户数据保存在 `data/user.txt`，上机记录保存在 `log/学号.txt`。

## 数据文件说明

- `data/user.txt`：存储所有学生的账号、密码、姓名、余额、状态等信息。
- `log/学号.txt`：每个学生的上机记录，包含日期、时间、时长、地点等。
